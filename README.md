## 📌 Project Overview

A modern and responsive **User Management System** built using **React.js** and **Tailwind CSS**.  
This application helps manage users efficiently with CRUD operations, email search, and status tracking.

---

## ✨ Key Features

- ✅ Create new users  
- ✅ View user list  
- ✅ Update user details  
- ✅ Delete users  
- ✅ Search users by email  
- ✅ Toggle user status (🟢 Active / 🔴 Inactive)  
- ✅ Fully responsive UI  
- ✅ Clean and minimal design

## ⚙️ Getting Started

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/mr-somu/UserManagement.git
```

### 2️⃣ Navigate to Project
```bash
cd UserManagement
```

### 3️⃣ Install Dependencies
```bash
npm install
```

### 4️⃣ Run the App
```bash
npm start
```
